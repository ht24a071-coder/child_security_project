# ゲームで学ぶ！防犯ゲーム

子供向けの防犯教育を目的としたアドベンチャーゲームです。
展示会等での試遊を想定し、タッチ操作や放置時の自動リセット機能を実装しています。

## 🛠 開発環境
- エンジン: Ren'Py
- 言語: Ren'Py Script (Pythonベース)
- エディタ: VS Code

## 📂 ファイル構成と役割
主要なコードは `game/` フォルダ内にあります。

| ファイル名 | 役割 | 備考 |
| --- | --- | --- |
| **script.rpy** | ゲーム本編のシナリオ | `label start:` (開始地点) はここにあります。 |
| **screens.rpy** | 画面デザイン (UI) | タイトル画面、プロフィール設定、ボタン配置など。 |
| **definitions.rpy** | データ定義リスト | 二つ名の単語リストや、アバター画像のリストを管理。 |
| **options.rpy** | ゲーム全体の設定 | 画面サイズ、音量、ビルド設定など。 |

## ✨ 主な実装機能

### 1. プロフィール設定画面 (`screens.rpy`)
- **二つ名スロット**: 形容詞・名詞を組み合わせてランダムまたは手動で名前を決定します。
- **アバター選択**: リストからアイコンを選択できます。
- **データ参照先**: 単語や画像のリストは `definitions.rpy` に記述し、モジュール化しています。

### 2. 展示用・放置対策 (`screens.rpy`)
- **screen inactivity_guard**: ゲーム中に一定時間（デフォルト30秒）操作がない場合、自動的にタイトル画面に戻ります。
- **実装場所**: `script.rpy` の `label start` 直後で `show screen` して常駐させています。

### 3. タイトル画面 (`screens.rpy`)
- `main_menu` スクリーンをカスタマイズ。
- 背景画像を表示し、Ren'Py側で「丸いボタン」や「ロゴ」を描画しています。

## 🚀 実行方法
1. Ren'Py Launcher を起動。
2. プロジェクト一覧から本プロジェクトを選択。
3. 「プロジェクトを起動」をクリック。
