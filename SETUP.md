# 子供安全プロジェクト - セットアップガイド

## 必要なライブラリ

このゲームでは、マイクを使った「大声を出す」ミニゲームのために追加のPythonライブラリが必要です。

### インストール方法

```bash
pip install sounddevice numpy
```

### ライブラリ一覧

| ライブラリ | バージョン | 用途 |
|-----------|-----------|------|
| sounddevice | 最新 | マイクからの音声入力を取得 |
| numpy | 最新 | 音量計算（振幅の計算） |

### 注意事項

1. **マイク権限**: Windows/Macでマイク使用許可が必要な場合があります
2. **フォールバック**: マイクが使えない場合は自動的に連打ゲームに切り替わります
3. **プライバシー**: 音声は録音・保存されません。音量レベルのみ検出します

### Ren'Pyへの組み込み

Ren'Pyのゲームディレクトリにある `python-packages` フォルダにライブラリを配置することで、配布版でも動作します。

```bash
# ゲームディレクトリで実行
pip install sounddevice numpy --target="game/python-packages"
```

## 開発環境

- Ren'Py 8.x 推奨
- Python 3.9+

## ゲームの起動

```bash
# Ren'Pyランチャーからプロジェクトを選択して起動
# または
renpy.exe child_security_project
```
